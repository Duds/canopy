# Product Context

## Project
Personal portfolio website for a Senior Web Designer

## Users
- Potential clients looking for brand/design services
- Creative directors reviewing candidates
- Other designers seeking inspiration

## Tone
Professional but approachable. Confident without arrogance. Visual-first, letting work speak louder than words.

## Anti-references
- Generic template portfolios
- Overly corporate/stiff language
- Cluttered layouts with too many elements
- Trend-chasing (glassmorphism, excessive gradients)

## Strategic Principles
1. **Show, don't tell** — Work samples front and center
2. **Editorial quality** — Typography and layout should demonstrate craft
3. **Restrained palette** — Warm neutrals with intentional accent moments
4. **Performance matters** — Fast, accessible, responsive
5. **Grid-aware** — Respect the grid but know when to break it intentionally

## Register
Brand (portfolio = design IS the product)
