Typography

Font Families

The system uses three complementary typefaces, each serving a distinct purpose:

Display Font — Syne

Design with purpose

Use for: Hero headlines, section titles, large display text. Always uppercase for headlines. Weights: 600–800.

Body Font — Space Grotesk

Space Grotesk is a proportional sans-serif typeface based on Space Mono. It brings a technical, contemporary feel while maintaining excellent readability across all sizes. Use for body text, navigation, buttons, and UI elements.

Use for: Body text, navigation, buttons, form elements, subheadings. Weights: 300–700.

Monospace Font — Space Mono

Brand Image Treatment

All images receive consistent treatment to maintain brand recognition:

  1. Subtle desaturation: Filter reduces saturation to 85% for cohesion
  2. Contrast boost: Slight contrast increase (1.05) for definition
  3. Terracotta overlay: Gradient multiply blend adds warmth
  4. Geometric accent: Optional corner triangle in primary accent
  5. Smooth scale: Images zoom slowly on hover (0.6s ease)

Usage Guidelines

Typography Hierarchy

  • Always use Syne in uppercase for headlines
  • Space Mono for all labels and metadata (uppercase, tracked)
  • Maintain adequate contrast between text sizes (≥1.25 ratio)
  • Maximum line length: 65–75 characters for body text

Color Usage

  • Primary accent (terracotta) for CTAs and key highlights only
  • Avoid pure black (#000) and pure white (#FFF)
  • Use bg-tertiary sparingly for section alternation
  • Maintain WCAG AA contrast ratios (4.5:1 minimum)

Spacing Rhythm

  • Section padding: 96px mobile, 128px desktop
  • Component gaps: 16–24px
  • Vary spacing to create visual rhythm—don't use same value everywhere
