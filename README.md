# canopy

Dale Rogers personal portfolio.

Built with Astro + Tailwind, deployed to Azure Static Web Apps.

## Local dev

```bash
npm install
npm run dev
```

## Content

Articles → `src/content/writing/`

See `_example.md` in each directory for frontmatter schema.
